get_compound_color#

fastf1.plotting.get_compound_color(compound, session)[source]#

    Get the compound color as hexadecimal RGB color code for a given
    compound.

    Parameters:

        - compound (str) – the name of the compound

        - session (Session) – the session for which the data should be
          obtained

    Return type:

        str

    Returns:

        A hexadecimal RGB color code
