get_compound_mapping#

fastf1.plotting.get_compound_mapping(session)[source]#

    Returns a dictionary that maps compound names to their associated
    colors. The colors are given as hexadecimal RGB color codes.

    Parameters:

        session (Session) – the session for which the data should be
        obtained

    Return type:

        dict[str, str]

    Returns:

        dictionary mapping compound names to RGB hex colors
