Results Data#

Results data is represented using the following objects. It can be
accessed through the Session.results attribute. To load a session, see
Loading Data.

  -------------------------------------------------- ---------------------------------------------------------------------------------------------------
  SessionResults(*args[, _cast_default_cols, ...])   This class provides driver and result information for all drivers that participated in a session.
  DriverResult(*args, **kwargs)                      This class provides driver and result information for a single driver.
  -------------------------------------------------- ---------------------------------------------------------------------------------------------------
