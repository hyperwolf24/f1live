list_team_names#

fastf1.plotting.list_team_names(session, *, short=False)[source]#

    Returns a list of team names of all teams in the session.

    By default, the full team names are returned. Use the short argument
    to get shortened versions of the team names instead.

    Parameters:

        - session (Session) – the session for which the data should be
          obtained

        - short (bool) – if True, a list of the shortened team names is
          returned

    Return type:

        list[str]

    Returns:

        a list of team names
