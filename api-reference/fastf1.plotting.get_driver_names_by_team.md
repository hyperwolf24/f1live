get_driver_names_by_team#

fastf1.plotting.get_driver_names_by_team(identifier, session, *, exact_match=False)[source]#

    Get a list of full names of all drivers that drove for a team in a
    given session based on a recognizable and identifiable part of the
    team name.

    Parameters:

        - identifier (str) – a recognizable part of the team name

        - session (Session) – the session for which the data should be
          obtained

        - exact_match (bool) – match the identifier exactly
          (case-insensitive, special characters are converted to their
          nearest ASCII equivalent)

    Return type:

        list[str]
