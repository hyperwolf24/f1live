get_driver_name#

fastf1.plotting.get_driver_name(identifier, session, *, exact_match=False)[source]#

    Get a full driver name based on the driver’s abbreviation or based
    on a recognizable and identifiable part of the driver’s name.

    Parameters:

        - identifier (str) – driver abbreviation or recognizable part of
          the driver name

        - session (Session) – the session for which the data should be
          obtained

        - exact_match (bool) – match the identifier exactly
          (case-insensitive, special characters are converted to their
          nearest ASCII equivalent)

    Return type:

        str
